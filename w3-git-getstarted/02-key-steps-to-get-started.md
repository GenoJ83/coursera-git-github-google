# Key Steps to Get Started

- Summary: Core steps to begin using Git in a project.
- Steps:
  1. Create a project folder.
  2. Navigate into the folder (`cd`).
  3. Run `git init` to start tracking the folder.
- Commands:
  - `git status` — show repository status
  - `git add <file>` — stage changes for commit
  - `git commit -m "message"` — record staged changes
- Tip: Commit frequently with clear messages to preserve history.
