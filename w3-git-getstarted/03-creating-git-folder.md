# Creating Git Folder

- Summary: How to create and prepare a project folder for Git.
- Commands:
  - `mkdir myproject` — make a directory
  - `cd myproject` — change into it
  - `git init` — initialize Git in the folder
- Explanation: `git init` creates a hidden `.git` directory that stores repository data.
- Windows note: enable "Show hidden files" to view `.git`, or use Git Bash commands.
