# Initialize Git

- Summary: What `git init` does and why it's needed.
- Effect: Creates the `.git` directory and starts tracking the project.
- Example:
  - `git init`
  - Output: `Initialized empty Git repository in /path/to/.git/`
- After init: use `git add` and `git commit` to record changes.
- Troubleshooting: If `git` is not found, install Git and add it to PATH; restart terminal.
