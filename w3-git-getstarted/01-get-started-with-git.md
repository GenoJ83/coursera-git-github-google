# Get Started with Git

- Summary: Quick intro to using Git after installation and configuration.
- Purpose: Track changes in project files and maintain history.
- Typical workflow: create a project folder, initialize a repo, add files, commit changes.
- Useful commands:
  - `mkdir myproject` — create project folder
  - `cd myproject` — enter folder
  - `git init` — initialize a new repository
- Notes: On Windows you can use "Git Bash Here" from Explorer to open a terminal in a folder.
